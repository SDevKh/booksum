# BookWise - AI Book Summarizer

BookWise is a modern AI-powered book summarizer web application. It allows users to upload/input book titles, request summaries generated by a background AI API, and store their collection in a personal library.

## Features

- **AI Summaries**: Enter a book title or drag/drop a document to analyze the book and get key takeaways, action items, and structural outlines.
- **Personal Library**: Save and manage summaries in your local catalog.
- **Fast Search & Filter**: Instant access to your summarized library.

## Getting Started

### Prerequisites

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Run Local Development Server

1. Start the backend summarization server (refer to `/backend` directory).
2. Start the frontend development server:
   ```bash
   npm run dev
   ```

### Tech Stack

- **Frontend**: Vite + React, Tailwind CSS, TanStack React Query, Lucide icons, Framer Motion.
- **Storage**: Clean, local browser LocalStorage (independent of external services).
